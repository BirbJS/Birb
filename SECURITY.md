# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.x.x   | :white_check_mark: |
| < 1.0.0 | :x:                |

## Reporting a Vulnerability

You can make vulnerability report via [huntr](https://huntr.dev/bounties/disclose). **Do NOT create issues to report vulnerability issues.**
